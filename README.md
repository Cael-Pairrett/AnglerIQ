## AnglerIQ 🎣

AnglerIQ is a fishing intelligence web application designed to help anglers make smarter decisions about where and when to fish. It combines weather data, forecasts, environmental conditions, and location insights to generate a “fishing outlook” for any freshwater fishing spot.

This project was built as a fun learning experience using Codex, where I explored “vibe coding” — rapidly building and iterating on a full-stack application with AI assistance. The goal was to take a real-world problem in the outdoors and turn it into a practical, data-driven tool.

### What it does
- Analyzes weather, wind, and environmental conditions
- Provides fishing outlook scores and predictions
- Displays forecasts based on future trends
- Shows map-based insights like boat ramps and access points
- Includes a built-in chatbot for fishing-related questions

### Why I built it
As someone interested in both analytics and the outdoors, I wanted to create something that actually solves a real problem — helping anglers decide if a trip is worth it before heading out.

This project also helped me learn how to:
- Build full-stack web applications
- Work with APIs and real-world data
- Design clean, user-friendly interfaces
- Use AI tools like Codex to accelerate development
